Site choice:

https://leadersandbest.umich.edu/

QA Checklist:

  1. Run through each page with the WAVE Chrome Extension.

  2. Users should be able to navigate through content using their keyboard.

  3. Users should be able to navigate content using a screen reader.

  4. The general architecture and hierarchy of the content should make sense.

  5. Charts and images should all have alt-text so that users with screen readers or users on a slow connection will still be able to understand the images.

  6. Decorative images should not be visible to screen readers.

Suggestions:

  1. The code could have better structure. It uses div from the beginning to the end which is very confusing. Things like nav, header and footer could be used to give it a better structure. Besides, all its code is in one file. There is no separate file for Javascript, CSS and so on. This makes it really hard for me to read its orginal code. Different file for styling and javascript could make it more easy to read.
  2. Have better accesibility. Currently, users cannot use keyboard to navigate through the website. Enable keyboard navigation could help people who have difficulty using mouse. Moreover, it can increase its usability by using landmark to help people who use assistive devices and do not want to browse the content linearly.
  3. Add alt for images and some alt should be more specific. Currently the website have five images which do not have alt text. For those that have alt text, the text is very simple. For example, there is text on the the website's one picture, but the text is not read by voiceover and the alt text is very brief which cannot fully illustrate the meaning of this picture.
  4. Add media queries to make the website responsive for different screen sizes. For now, if I make the web browser smaller, the website would not change and thus would only show part of it. Using media query could make it more responsive. In addition, making pictures responsive by setting their sizes correspond with screen size could also help.
